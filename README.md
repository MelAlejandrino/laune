# Laune - Personal Emotional Landscape

Laune is an elegantly designed, totally offline personal journaling and mood reflection application built with Flutter. Protect your most intimate thoughts behind native Biometric locks, visualize your emotional rhythms over time, and build a lasting habit of mindfulness.

## Core Features
* **Zero Cloud Architecture:** 100% data sovereignty. Your entries never leave your device.
* **Native Cryptography:** Seamlessly bind your journal to Apple FaceID or Android TouchID.
* **Heatmap Calendar:** Instantly visualize your daily moods with our gorgeous interactive emoji matrix.
* **Dynamic Analytics:** Automatic calculation of your streak and most frequent moods.

## Technology Stack
* **Framework:** Flutter (Dart)
* **Local Storage:** `shared_preferences`
* **Authentication:** `local_auth` (Secure Enclave integration)
* **Architecture:** `go_router` (Stateful Shell branches)

## Getting Started
To view this project locally on your machine:
```bash
# Clone the repository
git clone https://github.com/MelAlejandrino/laune.git

# Enter the directory
cd laune

# Install Dependencies
flutter pub get

# Run the app!
flutter run
```
